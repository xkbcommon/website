Detailed Description

Opaque struct representing an ISO 639-3 code (e.g.

"eng", "fra"). There is no guarantee that two identical ISO codes share the same struct. You must not rely on the pointer value of this struct.

See https://iso639-3.sil.org/code_tables/639/data for a list of codes.
